逻辑

逻辑本身是对推论和证明的思想过程， 可以分为三个部分，演绎法，归纳法， 溯因法

逻辑的本质

形式是逻辑的本质， 主要分为形式逻辑，非形式逻辑， 符号逻辑，数理逻辑

归纳

从观察中推论出广义化的过程

溯因

从推理到最佳解释的过程，理解为通过已知结果，得到来源的过程，由果溯因

演绎

在给定条件下，得到什么是可以得出的

机器学习

归纳是非常典型的机器统计学习方法， KNN, SVM

而演绎推理需要很多规则和事实，决策树尝试基于已知答案生成规则，与演绎推理非常相似

溯因推理类似与深度学习，在现实情况下，我们无法得到所有信息，深度学习通过实例数据在其网络上学习其模式，所以归类为溯因
