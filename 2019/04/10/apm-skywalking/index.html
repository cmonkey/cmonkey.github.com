apm-skywalking

skywalking

skywalking 是一个apm 系统， 在基于spring boot 2.1.4 , spring cloud Greenwich.RELEASE 实现的自定义网关中

通过添加skywalking agent , 将信息上报到backend 中，但报告错误为

错误的类型转换， 将一个Integer 类型转换为HttpResponseStatus

修改StatusInterceptor.beforeMethod 方法，判断了allArguments[0]参数的类型，并编译上传到服务器进行测试

不在抛出java.lang.ClassCastException: java.lang.Integer cannot be cast to io.netty.handler.codec.http.HttpResponseStatus 异常

但SkyWalking项目在issues 2126 中对支持spring gateway 放到6.1.0 版本中，所以暂时在6.0.0 版本中对gateway 没有支持，

所以需要在6.1.0 之后才能将SkyWalking 投入到基于spring gateway 的gateway 上

需要强调的是，目前对基于spring boot 2.1.4 版本的业务服务的支持是可以的，所以需要根据不同的情况选择是使用zipkin, 还是SkyWalking
