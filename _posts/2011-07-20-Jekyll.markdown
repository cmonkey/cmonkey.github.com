### 什么是Jekyll ?
   Jekyll 也许你没有听过，不过你应该知道github page,其使用的就是jekyll这个工具。
   它实际上是一个页面生成工具，每次根据你的文章，生成静态页面。

### Jekyll 获取

   Jekyll 源码托管在GitHub.com。
   你可以通过
    git clone git://github.com/mojombo/jekyll.git
   获取源码。

### Jekyll 安装

    gem install jekyll
    gem update --system
