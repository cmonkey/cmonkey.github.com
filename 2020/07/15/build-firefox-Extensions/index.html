build-firefox-Extensions

translate-man项目是一个浏览器翻译扩展项目，在firefox 中翻译体验比较好。

在前段时间作者将firefox 版本的插件在网站下架了，所以在新的firefox上不能使用

用过一段时间其它翻译插件后，还是认为此扩展更好用，所以需要看看如何自己编译并通过本地安装的方式安装此扩展

首先将项目clone 到本地，然后根据readme 文档的描述，执行项目编译

项目编译成功后会生成dist 目录，当dist 目录已经生成并可以看到文件已经正确的复制到了dist

那么可以根据 https://dev.to/scleriot/build-a-firefox-extension-step-by-step-5dbl 文档通过web-ext 执行打包，最后可以在dist 下的web-ext-artifacts 目录下找到打包好的xpi 文件

最后可以在firefox addons 下通过install add-on from file 的方式安装刚打好包的xpi 文件

通过以上步骤，终于可以使用translate-man, 真不容易
